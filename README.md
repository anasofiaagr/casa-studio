# Casa + studio — branco & azul

Modelo 3D interativo (three.js) da casa de 95,6 m² + garagem + studio num terreno de 15×30 m.

Abrir: https://anasofiaagr.github.io/casa-studio/

Botões: manhã / meio-dia / tarde (sol) · sem teto (ver planta por dentro) · legendas · medidas.
Arraste para orbitar, scroll/pinça para zoom. Norte ao fundo, leste à direita.
